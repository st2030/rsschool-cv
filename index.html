Solution to "Sum of Odd Numbers" kata using PHP: function rowSumOddNumbers($n) {
$num = 0;
for ($i = $n; $i > 0; $i--) {
$num += $i;
}
$sum = $num**2 - ($num - $n)**2;
return $sum;
}
